# 微信读书自动阅读脚本

轻量级微信读书脚本，自动阅读刷时间

## python运行

1. 使用 `uv` 管理项目
2. 运行 `main.py` 即可

## exe 运行

过程中会保存一个 `curl_config.sh` 文件，保存请求信息。

## 使用说明

点击开始按钮，按照提示操作即可
